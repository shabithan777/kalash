## Задание 1 
Генератор простых чисел. Просуммируйте возвращаемые числа.


## Описание проделанной работы
Разработка алгоритма генерации простых чисел
Были реализованы два подхода:

Подход 1: Замыкание
Создана внешняя функция prime_generator(), которая возвращает внутреннюю функцию get_next().

Внутри замыкания хранятся:

Список primes — уже найденные простые числа.

Переменная num — текущее проверяемое число.

Внутренняя функция is_prime() проверяет простоту числа, используя уже найденные простые числа для оптимизации.

При каждом вызове get_next() возвращается следующее простое число, состояние обновляется.

Подход 2: Генератор (yield)
Создана функция-генератор prime_generator(), которая использует yield для возврата простых чисел.

Бесконечный цикл while True генерирует простые числа одно за другим.

Состояние сохраняется автоматически между вызовами next().

3. Реализация суммирования
Написана функция sum_of_primes(n), которая:

Создаёт генератор простых чисел.

Последовательно получает n простых чисел с помощью next().

Накопливает их сумму в переменной total.

Возвращает полученную сумму.

4. Тестирование
Проведено тестирование для различных значений N:

N	Сумма первых N простых чисел	Проверка
1	2	✓
2	2 + 3 = 5	✓
5	2 + 3 + 5 + 7 + 11 = 28	✓
10	129	✓
20	639	✓
5. Оптимизация
Для проверки простоты используется оптимизированный алгоритм: проверка делимости только на простые числа, не превышающие квадратный корень из проверяемого числа.

Это значительно ускоряет работу по сравнению с перебором всех чисел от 2 до √n.
